# Fitness Daily Schedule Agent
This repository contains the code for my Kaggle Capstone Project: a multi-agent system that generates a personalized daily fitness, nutrition, and productivity schedule.

## Features
- Multi-agent architecture
- Fitness workout planner
- Meal planner (muscle gain focused)
- Daily schedule planner
- Memory system using JSON
- ipywidgets UI for interactive input
- Works on Kaggle Notebooks
